# SpoilGuard HTML Prototype (Sample)

This repository contains a **zero-setup HTML prototype** for the SpoilGuard project.  
It is provided as an instructional example for:

👉 **Semester Project Milestone 5: Rapid Prototyping & Draft Testing Plan**

---

## 📌 Purpose

This sample demonstrates how to translate earlier project work into a working prototype:

- Milestone 2 → Problem & Solution
- Milestone 3 → Requirements
- Milestone 4 → Use Cases & Scenarios
- Milestone 5 → Prototype + Testing Plan

The goal is to help you understand **how ideas become interactive designs**.

---

## 🎯 What This Prototype Demonstrates

The prototype includes flows for all four required use cases:

1. **UC-01: Scan Receipt (OCR Simulation)**
2. **UC-02: Inventory Management (My Fridge)**
3. **UC-03: Expiration Alerts**
4. **UC-04: Savings & Waste Analytics**

It also includes:
- Dashboard (hub navigation)
- Recipe suggestions
- Accessibility settings (high contrast, large font, audio alerts)

---

## 🚀 How to Run the Prototype

No installation required.

### Option 1 (Recommended)

## 🔄 Suggested Student Workflow

Use this sample as a **reference**, not a template to copy.

1. Review your Milestones 1–4  
2. Identify your **4 core use cases**  
3. Map each use case to **screen flows**  
4. Build a prototype (HTML, Figma, Forms, etc.)  
5. Demonstrate all flows clearly  
6. Create a testing plan (Milestone 6)
